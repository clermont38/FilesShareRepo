Files included:

MyRecipes1.jpg			- screen shot
MyRecipes1.odb			- Base database file contains macros to work with a switchboard main menu form
MyRecipes1DatabaseHelp.odt	- Write documentation file
MyRecipes1NoMacros.odb		- Base database file with no macros
ReadMe.txt			- this file

FoodPix - folder/subdirectory contains recipe picture files (.jpg) that link to Recipes table


Directions for use:

Unzip all files and subfolder, preferably in their own folder.
Open documents with appropriate LibreOffice application.

Suggestion: Recommend read the document file first.

Developed and tested to run in LibreOffice 4.4.2.2 installed on Windows 8.1 laptop.
